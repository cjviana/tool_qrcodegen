Sarting
=======
1. pip install virtualenv --install virtualenv package
2. virtualenv venv --create virtual environment
3. venv\Scripts\Activate --activate virtual environment
4. pip install pillow --install pillow package (Imaging Library) *** from venv ***
5. pip install qrcode --install qrcode package (QR COde Library) *** from venv ***


Finishing/Destroying
====================
1. deactivate
2. rm -r ENV
